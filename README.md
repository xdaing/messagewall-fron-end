# 留言墙前端

一个留言墙项目的前端部分

- [线上预览](http://message.gommie.cn/)
- [后端地址](https://github.com/xdaing/messagewall-back-end)

## 技术栈

vue3 + pinia + vue-router + axios + typescript + scss + vite

## 安装依赖

```sh
npm install
# or
yarn install
```

### 运行项目

```sh
npm run dev
# or
yarn dev
```

### 打包项目

```sh
npm run build
# or
yarn build
```
